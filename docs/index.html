Modules

Three tools. One mission.

Each module works standalone. Together, they give you complete visibility into what your agents do and why.

Module 01

Session Debugger

Record agent sessions transparently, then replay them step by step. See the full prompt, the model's reasoning, every tool call, and the cost of each decision.

  • ✓ Zero-config recording via hooks
  • ✓ Interactive step-through with keyboard controls
  • ✓ Full prompt/response inspection
  • ✓ Per-step cost and token tracking
  • ✓ Session diff — compare two runs side by side
  • ✓ Export to JSON, HTML, or Markdown

Module 02

Memory Inspector

Agent memory is the most underserved category in AI tooling. See how memory evolves, find stale entries, detect duplicates, and track what each agent knows over time.

  • ✓ Memory state visualization with health scoring
  • ✓ Diff between git commits or dates
  • ✓ Timeline view of memory evolution
  • ✓ Detect stale, orphaned, and duplicate entries
  • ✓ Framework-agnostic — works with any memory format
  • ✓ Actionable health recommendations

Module 03

Multi-Agent Test Harness

Pytest for AI agents. Define scenarios in YAML, mock tools, assert on outputs and costs, run as regression tests. No more "just run it and check manually."

  • ✓ YAML scenario definitions
  • ✓ Mock specific tools or agent responses
  • ✓ Assert on tool calls, outputs, memory, and cost
  • ✓ Use recorded sessions as test fixtures
  • ✓ Regression detection across code versions
  • ✓ CI-friendly exit codes and structured output
# tests/customer-support.yaml
name: "Customer refund flow"
agents:
  - role: support-agent
    model: claude-sonnet-4-6
scenario:
  - user: "Refund order #12345"
  - assert:
      tool_called: lookup_order
      cost_under: 0.05
  - mock_tool_response:
      lookup_order: { status: delivered }
  - assert:
      response_contains: "refund"
How It Works

Debug in 60 seconds

1

Install

One command. No accounts, no config, no dashboards.

npm i -g agentlens
2

Record

Wrap your agent command. It runs normally — recording is transparent.

alens record claude "task"
3

Replay

Step through every decision. See prompts, responses, tool calls, and costs.

alens replay ses_abc123
4

Test

Write YAML scenarios. Assert on behavior. Run as regression tests.

alens test run flow.yaml
